# Personal Site

A lightweight personal website and blog built with plain HTML, CSS, and JavaScript.

## Publishing

This repository is ready for GitHub Pages. In the repository, open **Settings → Pages**, choose **Deploy from a branch**, select the default branch and `/ (root)`, then save.

## Adding a post

1. Copy `posts/why-i-am-writing.html`.
2. Rename it with a URL-friendly title.
3. Replace the title, date, introduction, and article body.
4. Add a link to it from `index.html`.
